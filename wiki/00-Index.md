# PostDesk Knowledge Base

Welcome to the **PostDesk Memory Bank** – an Obsidian-style, AI-friendly knowledge base designed to reduce cognitive load and prevent brain-rot for human product managers, interns, and AI coding agents.

This knowledge base uses clear, dense, un-slop structures intended for high-speed indexing by LLMs and humans alike.

## Getting Started
If you are an AI Coding Agent acting within this codebase, you MUST read the `02-standards/03-Coding-Conventions.md` and `04-tasks/*` logs to orient yourself to your current objectives before writing ANY code.

If you are an intern, review `01-architecture/01-Architecture-Concepts.md` and `02-standards/01-Tool-Stack.md` to understand the domain primitives.
